Scanning…

VL models are detected automatically by filename keywords (llava, moondream, minicpm-v, …). Set the matching mmproj file above to enable image input for those models. The vision model dropdown in the chat header also auto-detects mmproj files. If no VL model is set, BLIP auto-captioning is available as a fallback.

⚠️ API Cost Notice Mimir makes one extra LLM call per message for memory curation — deciding what to remember and how to tag it. When using paid APIs (OpenAI, Anthropic, Google), this will roughly double your per-message token costs.

💡 Free alternative: Use open-source models via Ollama or a Local GGUF for unlimited free usage with no API costs.
